# ERCOT Scarcity Pricing Analysis

## Dependencies
* R 4.1.1
* Stata SE or greater (for execution from console with RStata)

## To Run
1. Clone repository
2. Open R_files/ercot_incentive_analysis.Rmd
3. Set stata relative options in ercot_incentive_analysis.Rmd and relative file path in energy_time_2to1_full.do
4. Set boolean for appendix run (default to false). Appendix results take ~18hrs to complete, depending on RAM.
